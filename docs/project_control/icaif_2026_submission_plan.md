# ICAIF 2026 Submission Plan

## Hard Constraints

- Deadline: August 2, 2026 AOE.
- Venue: ICAIF 2026, Milan, November 14-17, 2026.
- Limit: 8 pages total, including figures and references.
- Review: double blind, no appendix or supplementary material.
- Strategy: benchmark plus directed network stress transmission, with GNN only
  as a predictive extension if it is materially strong.

## Paper Scope

Main paper:

- Provenance-aware multi-layer event dataset.
- Timestamp-aligned lead-lag evidence.
- VAR/FEVD or rolling VAR spillover table.
- Network contagion map.
- Robustness summary.
- Prediction table only if baselines and labels are ready.

Repo-only:

- Extended event diagnostics.
- Full robustness grids.
- Hawkes and TE sensitivity details.
- GNN ablations if not paper-grade.

## Figure Budget

- Coverage heatmap.
- Event-time stress map.
- Directed contagion network map.
- Compact directed-edge table.
- Prediction/GNN metric table only if strong.
